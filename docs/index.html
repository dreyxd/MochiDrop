Get MochiDrop up and running in minutes

1

Clone & Install

cd mochidrop
pip install -r requirements.txt
2

Configure Environment

cp .env.example .env
# Edit .env with your configuration

Initialize Database

4

Run the Bot

üë• User Registration

How users can register and get started with MochiDrop

Start the Bot

Users begin by sending /start to the MochiDrop bot on Telegram.

/start

Automatic Registration

The bot automatically creates a user profile with their Telegram information.

  • Telegram ID stored securely
  • Username and first name captured
  • Role set to 'receiver' by default

Wallet Registration

Users provide their Solana wallet address to participate in airdrops.

üëë Admin Setup

Configure admin access and permissions

1. Environment Configuration

Add admin Telegram IDs to your .env file:

# Admin Configuration
ADMIN_TELEGRAM_IDS=123456789,987654321
ADMIN_PASSWORD=your_secure_password_here

2. Admin Authentication

Admins must authenticate using the /admin command:

1

Send /admin to the bot

2

Enter the admin password when prompted

3

Access granted - admin dashboard appears

3. Security Features

Role-Based Access

Only authorized Telegram IDs can access admin functions

Password Protection

Additional password layer for admin authentication

Encrypted Storage

All admin wallet private keys are encrypted at rest
